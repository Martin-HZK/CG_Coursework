# Computer Graphics Coursework

## KeyBoard Shortcuts

### Universal

* `F`: Zoom out
* `R`: Zoom in
* `L`: Change Light Type
* `C`: Change Camera Type


### ModelViewerCamera ONLY
* `&uaar;`: Camera Up 
* `&daar;`: Camera Down
* `&rarr;`: Camera Right
* `&larr;`: Camera Left
* `Y`: Start Auto Orbit Revolution
