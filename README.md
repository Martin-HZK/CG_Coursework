# Computer Graphics Coursework

## KeyBoard Shortcuts

### Universal

* `F`: Zoom out
* `R`: Zoom in
* `L`: Change Light Type
* `C`: Change Camera Type
* `N`: Turn off all the lights

### ModelViewerCamera ONLY
* `↑`: Camera Up 
* `↓`: Camera Down
* `←`: Camera Right
* `→`: Camera Left
* `Y`: Start/End Auto Orbit Revolution
